# JavaWebServer

使用Java编写的Web服务器，主要提供了简单的HTTP服务、Servlet应用服务与模板引擎

An web server written in JAVA, mainly provides simple HTTP connector, servlet container,  and template engine.

## 功能

- 基础HTTP功能
  - HTTP协议解析
  - HTTP协议处理
